衆議院選挙 小選挙区当選者予測します
====

## はじめに

小選挙区選挙での当選者を予測します. 

名前でググった時の検索ヒット数が人気を反映, つまり投票数を反映していると考えます. 

ここから, 当選者の予測が出来るのでは? 

## 方法

- 候補者リストを作って, 

- 名前でググって検索ヒット数を集めて, 

- 上位者を当選者と予想してみます. 


## 進捗

候補者リストを作りました. 1492人います. 

ヒット数を調べました. 

あとは誰が当選するかを確認するだけ. 

## 内容

MacOSXで動作しています. 

- downloadHtml.bash 候補者リストhtmlのダウンロード

- scrape.bash htmlをパースして候補者リストをつくる

- kouho.list 選挙区と名前(と政党と年齢)のデータテーブル

- kouho.hit.list kouho.list + 検索ヒット数のデータテーブル
